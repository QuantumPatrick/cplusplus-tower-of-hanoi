# cplusplus-tower-of-hanoi
A graph search algorithm for Tower of Hanoi (3 rods and 3 disks)

## Rules of the Tower of Hanoi
* First, each move allows one disk to change position. No more, no less.
* Second, only the top disk of any "tower" may be moved.
* Third, no larger disk may be placed on top of a smaller disk.
